# Install & Run
1. Make sure that you have installed the [Node](https://nodejs.org/) LTS on your machine
2. Install dependencies with the following command:
```shell
npm install

# OR

yarn add
```
3. Run the app with the following command
```shell
npm run dev

# OR

yarn dev
```
